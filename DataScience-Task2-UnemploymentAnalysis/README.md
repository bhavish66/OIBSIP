# Unemployment Analysis with Python

## Overview
This project analyzes unemployment trends in India using Python. It explores regional and monthly unemployment patterns and compares unemployment rates before and after the COVID-19 pandemic.

## Dataset
- Unemployment in India Dataset (Kaggle)

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Features
- Data loading and cleaning
- Null value handling
- Date conversion
- Region-wise unemployment analysis
- Month-wise trend analysis
- Time series visualization
- Top 10 states with highest unemployment
- Correlation heatmap
- Pre-COVID vs Post-COVID comparison

## Project Structure
- Unemployment_Analysis.ipynb
- README.md
- Screenshots

## Outcome
The project highlights unemployment trends and the impact of COVID-19 across different regions of India.
