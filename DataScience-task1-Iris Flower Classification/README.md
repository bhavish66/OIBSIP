# Iris Flower Classification

## Overview
This project builds a machine learning classification model to identify the species of an iris flower (Setosa, Versicolor, or Virginica) based on its physical measurements.

## Dataset
- Iris Dataset (Built-in from scikit-learn)

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## Features
- Data loading and preprocessing
- Exploratory Data Analysis (EDA)
- Pairplot and Boxplot visualizations
- Feature selection
- Train-test split
- Logistic Regression model
- K-Nearest Neighbors model
- Model evaluation using Accuracy, Confusion Matrix, Precision, Recall and F1-Score
- Best model comparison

## Project Structure
- Iris_Flower_Classification.ipynb
- README.md
- Screenshots

## Outcome
The best-performing classification model was selected based on evaluation metrics.
