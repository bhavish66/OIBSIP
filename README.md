OIBSIP – Data Science Internship

Overview

This repository contains the projects completed as part of the Oasis Infobyte Data Science Internship. Each project focuses on applying data science, machine learning, and data analysis techniques using Python. The projects cover classification, regression, exploratory data analysis (EDA), and natural language processing (NLP), demonstrating the complete workflow from data preprocessing to model evaluation and visualization.

Projects

Task 1 – Iris Flower Classification

Objective:
Build a machine learning classification model to predict the species of an iris flower using its physical measurements.

Dataset

* Iris Dataset (Built into Scikit-learn)

Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Jupyter Notebook

Key Features

* Data loading
* Exploratory Data Analysis (EDA)
* Pairplot and Boxplot visualizations
* Feature selection
* Logistic Regression
* K-Nearest Neighbors (KNN)
* Model evaluation using Accuracy, Confusion Matrix, Precision, Recall, and F1-Score
