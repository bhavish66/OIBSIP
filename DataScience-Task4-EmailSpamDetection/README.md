# Email Spam Detection with Machine Learning

## Overview
This project develops a Natural Language Processing (NLP) model to classify messages as spam or legitimate (ham) using machine learning techniques.

## Dataset
- SMS Spam Collection Dataset (Kaggle)

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLTK
- Matplotlib
- Seaborn
- Jupyter Notebook

## Features
- Data loading
- Text preprocessing
- Lowercase conversion
- Punctuation removal
- Stopword removal
- Stemming
- TF-IDF Vectorization
- Train-test split
- Multinomial Naive Bayes
- Logistic Regression
- Model evaluation using Accuracy, Precision, Recall, F1-Score and Confusion Matrix
- WordCloud visualization

## Project Structure
- Email_Spam_Detection.ipynb
- README.md
- Screenshots

## Outcome
The project successfully classifies spam and legitimate messages using NLP and machine learning algorithms.
