# Car Price Prediction with Machine Learning

## Overview
This project predicts the selling price of used cars using machine learning regression models based on various vehicle attributes.

## Dataset
- Vehicle Dataset from CarDekho (Kaggle)

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## Features
- Data cleaning
- Duplicate removal
- Feature engineering
- Brand extraction
- Car age calculation
- Exploratory Data Analysis
- Label Encoding
- Correlation heatmap
- Linear Regression
- Random Forest Regression
- Model evaluation using MAE, RMSE and R² Score
- Feature importance visualization

## Project Structure
- Car_Price_Prediction.ipynb
- README.md
- Screenshots

## Outcome
Multiple regression models were trained and compared to identify the best-performing model for predicting used car prices.
